环境准备
--------

1.Python2.7.13 虚拟环境(virtualenv)

    Windows Python虚拟环境安装：
        pip install virtualenv
    Windows Python虚拟环境创建：
        virtualenv venv
    进入/启动Python虚拟环境：
        venv/bin/activate
    退出Python虚拟环境：
        deactivate

2.sphinx和所需第三方包安装：
    pip install sphinx sphinx-autobuild sphinx_rtd_theme
