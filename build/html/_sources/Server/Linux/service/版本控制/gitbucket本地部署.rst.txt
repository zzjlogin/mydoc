gitbucket本地部署
====================================================================

下载软件包
-----------------------------------------

.. code-block:: bash 

    [root@centos-7 src]# wget https://github.com/gitbucket/gitbucket/archive/4.21.2.tar.gz

运行服务
---------------------------------------------------

.. code-block:: bash 

    [root@centos-150 src]# java -jar gitbucket.war
