环境准备
====================================================================

安装python
----------------------------------------------

anaconda下载python_

.. _anaconda下载python: https://www.anaconda.com/download/

.. note:: 安装过程请勾选添加到环境变量的选项，不然后续的命令没法直接使用。

ipython的安装
----------------------------------------------

.. code-block:: text 

    C:\Users\Administrator>pip install ipython

.. note:: 如果你感觉ipython不好用，可以换用bpython。

bpython的安装_

.. _bpython的安装: http://www.cnblogs.com/zhaojiedi1992/p/zhaojiedi_python_006_bpython.html
