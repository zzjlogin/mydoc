xpinyin作用
======================================================================================================================================================

把汉字转换为拼音


样例：

.. code-block:: python
    :linenos:

    from xpinyin import Pinyin
    p = Pinyin()
    print(p.get_pinyin(u"上海", show_tone_marks=True))  #标记声调
    print(p.get_initial(u"上")) #一个汉字的拼音首字母大写
    print(p.get_initials(u"上海")) #每个汉字的拼音首字母大写
