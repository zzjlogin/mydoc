======================================================================================================================================================
dict字典创建DataFrame数据对象
======================================================================================================================================================


:Date: 2019-3

.. contents::


字典value是字符串/数值
======================================================================================================================================================

.. code-block:: python
    :linenos:

    In [17]:  d = {'a':123, 'b':321,}
    In [19]: data = pd.DataFrame([d])
    In [20]: data
    Out[20]:
        a    b
    0  123  321


字典value是list/tuple
======================================================================================================================================================

.. code-block:: python
    :linenos:

    In [29]: d2 = {'a':[123, 234,], 'b':[211, 221],}
    In [30]: data = pd.DataFrame(d2)
    In [31]: data
    Out[31]:
        a    b
    0  123  211
    1  234  221
