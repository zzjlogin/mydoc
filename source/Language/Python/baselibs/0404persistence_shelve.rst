数据持久化
======================================================================================================================================================


shelve
======================================================================================================================================================

shelve是一个持久的，类似字典的对象。

.. code-block:: python
    :linenos:

    In [21]: import shelve

    # 打开一个文件，可以不存在，会创建的
    In [22]: d=shelve.open("a.shelve")

    # 设置一些属性值
    In [23]: d["name"]="zzjlogin"

    In [24]: d["age"]= 26

    # 列出keys
    In [26]: list(d.keys())
    Out[26]: ['name', 'age']

    # 关闭
    In [27]: d.close()

    # 再次打开
    In [28]: d=shelve.open("a.shelve")

    # 还是可以获取到原有的数据的
    In [29]: d["name"]
    Out[29]: 'zzjlogin'
