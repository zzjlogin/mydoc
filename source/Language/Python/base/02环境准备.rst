安装python
------------------------------------------------------------------------------------------------------------------------------------------------------

.. hint::

    本安装实例是安装anaconda这个发行版Python，这个版本的优点是：
        - 很多方便优秀的第三方库已经默认安装。
        - 集成了ipython notebook
        - 安装简单，易用。
        - 科学计算中尤其方便。

`anaconda下载python <https://www.anaconda.com/download/>`_

.. note:: 安装过程请勾选添加到环境变量的选项，不然后续的命令没法直接使用。
