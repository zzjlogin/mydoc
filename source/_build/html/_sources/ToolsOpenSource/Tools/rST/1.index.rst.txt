reST介绍
===================

reStructuredText是一种轻量级的文本标记语言，这种标记语言比Markdown强大很多，支持多种排版样式。
不过这里只介绍开发人员主要会用到的一些样式。
reST文档的详细介绍见http://docutils.sourceforge.net/rst.html
