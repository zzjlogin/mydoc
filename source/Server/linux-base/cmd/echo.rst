    :linenos:

                        能否引用变量  |     能否引用转移符          |  能否引用文本格式符(如：换行符、制表符)

.. _echo-args:

参数
======================================================================================================================================================

\-n
    - 输出字符最后不加换行符，默认会自动加换行符。
    - 示例：

.. code-block:: bash
    :linenos:

    [root@centos6 ~]# echo 'test'
    test
    [root@centos6 ~]# echo -n 'test'
    test[root@centos6 ~]# 


-e
    - 用反斜线转移 ``\``
    - 示例：

.. code-block:: bash
    :linenos:
    
    [root@centos6 ~]# echo -e "OK! \c"
    OK! [root@centos6 ~]# echo OK! \c    
    OK! c
    [root@centos6 ~]# echo -E "OK! \c"
    OK! \c
    [root@centos6 ~]# echo "OK! \c"   
    OK! \c
    [root@centos6 ~]# echo -e "OK! \c"
    OK! [root@centos6 ~]# 

-E
    - 禁用反斜线转移 ``\`` ，默认禁用
    - 示例：

.. code-block:: bash
    :linenos:
    
    [root@centos6 ~]# echo -E OK! \c
    OK! c
    [root@centos6 ~]# echo "OK! \c"   
    OK! \c
    [root@centos6 ~]# echo -e "OK! \c"
    OK! [root@centos6 ~]# 

--help
    - 显示帮助并退出

--version
    - 显示版本信息
