/etc/passwd文件中的第10个用户的ID

.. code-block:: bash
    :linenos:

    [root@zzjlogin ~]# cat /etc/passwd |head -n 10 |tail -n 1 |cut -d ":" -f 3
    10

计算/etc/passwd文件中的第10个用户和第20用户的ID之和 

.. code-block:: bash
    :linenos:

    cat /etc/passwd |awk -F ":" ' {if(NR==10 || NR==20 ){sum+=$3}} END{print sum}'
