显示链接情况
------------------------------------------------------------------------------------------------------------------------------------------------------

显示正连接本主机的每个远程主机的IPv4地址和连接数，并按连接数从大到小排序 

.. code-block:: bash
    :linenos:

    [root@zzjlogin ~]# ss -tan |grep "^ESTAB" |egrep "[0-9]{1,3}(\.[0-9]{1,3}){3}:[0-9]{1,6}[[:space:]]*$" -o |cut -d ":" -f 1 |sort |uniq -c | sort  -r -n -t " " -k1
        1 192.168.161.1
