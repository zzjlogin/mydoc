Linux命令所在安装包/卸载安装
======================================================================================================================================================



Linux命令所在安装包
======================================================================================================================================================

.. code-block:: bash
    :linenos:

    which vi
    whereis vim
    rpm -qf /usr/bin/vim

    rpm -qa | grep vim
    
    yum search ifconfig

Linux安装包卸载
======================================================================================================================================================

.. code-block:: bash
    :linenos:

    yum remove [程序名]

    rpm -e [package_name]


--nodeps选项来忽略依赖，但这可能导致系统不稳定或者其他软件出现问题：


.. code-block:: bash
    :linenos:

    rpm -e --nodeps example-package
