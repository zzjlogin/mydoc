======================================================================================================================================================
nginx添加模块
======================================================================================================================================================


整体步骤：
    - 用命令查看现在的安装参数 ``nginx -V``
    - 下载对应版本的nginx源码包，然后解压。
    - 运行 ``./configure`` 后面参数指定和原来相同的基础上加上要添加的模块。
    - 运行 ``make``
    - 然后用生成的文件 ``nginx`` 替换系统原来的nginx二进制文件。

.. note::
    上面不能 ``make install`` ，否则会覆盖原来的程序。
