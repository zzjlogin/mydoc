MySql备份恢复
======================================================================

MySQL客户端备份MySQL数据库
======================================================================

MySQL8.0及以后-备份命令
----------------------------------------------------------------------

mysqldump -uroot -pxxx -h192.168.19.20 -P3336 --databases chargedata >d:\chargedata.sql


--databases
	备份文件有数据库创建指令，导入时不用创建数据库，直接导入即可。


MySQL8.0及以后-恢复命令
----------------------------------------------------------------------

mysql -uroot -pxxx <chargepara.sql
