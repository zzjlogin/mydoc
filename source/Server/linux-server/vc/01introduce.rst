介绍
======================================================================================================================================================


版本控制最主要的功能就是追踪文件的变更。

版本控制包括：
    - 检入检出控制
    - 分支和合并
    - 历史记录
    

开源版本控制工具
======================================================================================================================================================

- SVN
- git
- CVS
- RCS
- SCCS

其中svn和git是现在流行的版本控制器
