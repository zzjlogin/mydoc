svn存储版本数据也有2种方式
    - BDB(一种事务安全型表类型)和FSFS(一种不需要数据库的存储系统)。因为BDB方式在服务器中断时，有可能锁住数据，所以不太安全。
    - FSFS,比BDB方式更安全一点。
