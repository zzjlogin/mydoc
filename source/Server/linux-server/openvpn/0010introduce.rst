OpenVPN特点
------------------------------------------------------------------------------------------------------------------------------------------------------


OpenVPN特点：
    - 采用CS架构，需要安装OpenVPN客户端；
    - 采用加密传入；
    - 认证方式：密钥文件、密码可以结合使用
    - 认证可以结合openldap/pam/mysql认证。
    - OpenVPN是一个SSL VPN，因此并不与IPSec、L2TP或PPTP兼容
