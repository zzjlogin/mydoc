zabbix版本说明
======================================================================================================================================================

官方关于软件周期的规划：
    发布周期：https://www.zabbix.com/cn/life_cycle_and_release_policy
    软件版本路线：https://www.zabbix.com/cn/roadmap

zabbix软件分为两种：
    - 标准版：维护周期短迭代快的版本，一般维护周期6个月；
    - LST版本：LST维护周期3-5年的长支持版本。

现在的稳定版：
    2.2、3.0、4.0，2020年发布5.0

常见的标准版：
    3.4.x
