:Date: 2019-3

.. contents::



删除nan相关行/列
======================================================================================================================================================

1. 一列中所有元素都是nan，则删除这一列：

.. code-block:: python
    :linenos:

    df.dropna(axis=1,how='all')

2. 一行中所有元素都是nan，则删除这一行：

.. code-block:: python
    :linenos:

    df.dropna(axis=0,how='all')

3. 删除表中含有任何NaN的行

.. code-block:: python
    :linenos:

    data.dropna(axis=0,how='any')



4. 根据某一列，如果这一列值是nan，则删除这一行

df.dropna(axis=0, subset=['email'], inplace=True)
