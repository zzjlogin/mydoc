tarfile
======================================================================================================================================================

tarfile提供打包归档

使用方法和zipfile基本一样。只是打开模式上，有写不同。

.. csv-table:: 
   :header: "模式","描述"
   :widths: 30,50

    "r|*",	    打开压缩流通用压缩
    "r|",	    打开压缩
    "r|gz",	    打开gzip流读取
    "r|bz2",    打开bz2流读取
    "r|xz",	    打开xz流读取
    "w|",	    通用压缩写入
    "w|gz",		打开gzip流写入
    "w|bz2",	打开bz2流写入
    "w|xz",     打开xz流写入
