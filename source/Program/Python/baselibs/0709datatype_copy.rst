copy
======================================================================================================================================================

此模块提供深copy和浅copy功能

样例： 

.. code-block:: python 

    In [113]: class Point:
        ...:     pass
        ...:

    In [114]: p= Point()

    In [116]: import copy

    In [117]: p2 =copy.copy(p)

    In [119]: p3 = copy.deepcopy(p)
