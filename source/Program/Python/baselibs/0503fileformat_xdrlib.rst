xdrlib详解
======================================================================================================================================================

支持外部数据表示标准,提供encode和decode xdr数据。

样例： 

.. code-block:: python
    :linenos:

    import xdrlib
    p = xdrlib.Packer()
    try:
        p.pack_double(8.01)
    except xdrlib.ConversionError as instance:
        print('packing the double failed:', instance.msg)
